# Fotos do Hero — Login Slideshow

## Como substituir pelas fotos do Instagram

1. Abra https://www.instagram.com/brigadacamarao/
2. Clique na foto desejada → Clique com botão direito → "Salvar imagem como..."
3. Salve nesta pasta com os nomes:
   - `hero-1.jpg` — Brigadistas em ação / atendimento
   - `hero-2.jpg` — Equipe uniformizada / treinamento
   - `hero-3.jpg` — Evento grande (EXPOMINAS, festival, etc.)
   - `hero-4.jpg` — Primeiros socorros / resgate
   - `hero-5.jpg` — Equipe reunida / foto de grupo

### Dica: Use fotos em alta resolução (mínimo 1200px de largura)

As fotos aparecem com leve blur e gradient overlay, então fotos escuras ou com muitos detalhes funcionam melhor.

### Alternativa rápida (via InstaLoader):
```bash
pip install instaloader
instaloader --no-videos --no-captions --no-metadata-json brigadacamarao
```
Depois copie as 5 melhores fotos para cá.
